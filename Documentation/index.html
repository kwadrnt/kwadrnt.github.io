Unica

Personal One page HTML5 Template

General

Layout

This template has responsive layout. There are three layout modes related to the device width.

  • Desktop - more than 1200px
  • Tablet - between 767px and 1200px
  • Phone - less then 767px

HTML

The general template structure is the same throughout the template. Here is the general structure.

<!-- BEGIN NAV -->
<div class="main-nav">
    ...
</div>
<!-- END NAV -->

<div class="wrapper h-card">

    <!-- BEGIN HEADER -->
    <header class="main-header">
        ...
    </header>
    <!-- END HEADER -->

    <!-- CONTENT SECTIONS -->

    <!-- BEGIN FOOTER -->
    <footer class="main-footer">
        ...
    </footer>
    <!-- END FOOTER -->

</div>

By default, there are seven sections.

CSS

Template includes three main CSS files.

<link rel="stylesheet" href="styles/animate.css">
<link rel="stylesheet" href="styles/normalize.css">
<link rel="stylesheet" href="styles/main.css">

Animate.css is a bunch of cool, fun, and cross-browser animations for you to use in your projects.

Normalize.css makes browsers display elements more consistently and in line with modern standards. It's good alternative to old CSS resets.

The main.css file contains all of the specific stylings for the page.

Also there are several addition css files for third-party plugins.

Sections

Each site section has common structure.

<section id="section-id" class="section">
    <div class="section-inner">

        <div class="container">
            <!-- Section content -->
        </div>

    </div>
</section>

Use the code to create additional section. Then add a new link to the main navigation with same href value as the section id.

Section background

There are several ways to change sections background. You can use two predefined color backgrounds (light and dark) or add image background.

If you would like to add color background to the section you need to add "section-bg-light" or "section-bg-dark" css class to it.

<section id="about" class="section section-bg-light">
    <div class="section-inner">
        …
    </div>
</section>

If you want to add image background you need to add "section-bg-img" css class to the section and insert div with a class of "bg" inside.

<section id="about" class="section section-bg-img">
    <div class="bg"></div>
    <div class="section-inner">
        …
    </div>
</section>

Then you need to add background image to the section via css:

#about bg {
    background-image: url('../images/about-bg.jpg');
}
You can't add background image to the section, if previous section already have one and both sections have not rectangular shape.

Section shape

By default, all of the sections have rectangular shape. You can change shape of the section using four additional css classes.

  • section-top-rise
  • section-top-fall
  • section-bottom-rise
  • section-bottom-fall

To change section layout simply add class to the section element. Like so:

<section id="about" class="section section-bg-light section-top-rise section-bottom-rise">
    <div class="section-inner">
        …
    </div>
</section>

Grid System

Template has simple grid system. You can use up to 6 columns. The system relies on .rows and .spans. You can either attach the classes to your HTML elements or re-purpose their properties to allow for more semantic markup.

How to use

First create a row.

<div class="row"></div>

Add the selector corresponding to the desired width of your element like so, where the number refers to the amount of columns you would like your element to span.

<div class="row">
    <div class="span_2"></div>
</div>

This would create an element two columns wide. The required presentational classes can also be added to existing classes:

<div class="article span_8"></div>

Skills

To set up your skills, you need to change the "data-progress" attribute of the each div with a class of "skill-item".

<div data-progress="90" class="skills-item">
    <div class="skills-item-progress"></div>
    <h4>Identica</h4>
</div>

Portfolio

To link the project with the filter, you need to fill the "data-groups" attribute. It has to be a valid JSON array of strings.

<li class="works-item" data-groups='["branding"]'>
    ...
</li>

You can define multiple groups.

<li class="works-item" data-groups='["branding", "print", "identica"]'>
    ...
</li>

Instagram

To display your instagram photos, you need to fill "data-user-id" and "data-access-token" attributes of the div with an id of "instafeed". Click here to get your user id and token.

<div id="instafeed" data-user-id="YOUR-USER-ID" data-access-token="YOUR-ACCESS-TOKEN" class="instafeed"></div>

If you want to display images tagged with #somehashtag you need to fill "data-client-id" and "data-hashtag" attributes and leave "data-user-id" and "data-access-token" attributes empty. You'll need to get going is a valid client id from Instagram's API. You can easily register for one on Instagram's website.

<div id="instafeed" data-client-id="YOUR-CLIENT-ID" data-hashtag="HASHTAG" class="instafeed"></div>

Tweets

To display your last tweets, you need to do several steps.

  • Go to www.twitter.com and sign in as normal, go to your settings page.
  • Go to "Widgets" on the left hand side.
  • Create a new widget for what you need eg "user time line" or "search" etc.
  • Feel free to check "exclude replies" if you don't want replies in results.
  • Now go back to settings page, and then go back to widgets page and you should see the widget you just created. Click edit.
  • Look at the URL in your web browser, you will see a long number like this: 345735908357048478
  • Write this number in the "data-id" attribute of the div with an id of "tweets".
<div id="tweets" class="tweets" data-id="your-widget-id"></div>

Contact form

To make contact form work properly, you need to change address of the recipient in the send.php file.

$recipient = "demo@alexilin.ru"; // Replace this address with yours.

Color schemes

Template comes with seven additional color schemes. They are placed in "styles/schemes" folder. To select a different color, you need to add the appropriate css file to the end of <head> section and remove main.css.

<link href="styles/schemes/scheme-2.css" rel="stylesheet">

We also added colos.scss file with which you can quickly create an infinite number of new color schemes using Sass.
